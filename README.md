<h1>Учебный Django проект</h1> Используется sqlite для базы данных. Реализована регистрация, авторизация, отзывы, заказ товаров, каталог с фильтрами и пагинацией, админ, дополнительные функции на страницах для админа а также отдельная дамин панель. Html, Css, Js стили на каждой странице, прелоадер, анимации.
